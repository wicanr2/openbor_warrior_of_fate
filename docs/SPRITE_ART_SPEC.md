# 角色分鏡素材規格

本文件給後續接手繪製、外包或匯入角色圖的工作者使用。請在修改任何 GIF 前先閱讀。

## 透明底色：洋紅色（Magenta key）

目前 Warriors of Fate 的角色 GIF 使用**洋紅色調色盤鍵色**，不是 GIF 的 alpha 透明通道。

| 項目 | 規格 |
| --- | --- |
| 鍵色 | RGB `#FF00FF`（255, 0, 255） |
| 用途 | 角色輪廓以外的背景；OpenBOR 以此色視為透明 |
| 檔案格式 | 保持 GIF（建議索引色／8-bit palette） |
| 不可使用 | 漸層洋紅、接近但不完全相同的粉紅色、預先合成的背景 |

> GitHub 的圖片預覽會把 `#FF00FF` 顯示成洋紅底；遊戲引擎中它是透明區域。README 的分鏡總覽為了易讀，會在產生預覽時把 `#FF00FF` 暫時轉成透明，但這**不代表**原始 GIF 應改用 alpha。

## 製作與匯出規則

1. 從對應角色資料夾取出目標 GIF，**保留原始畫布寬高**。
2. 將角色以外的每個背景像素填為完全相同的 `#FF00FF`。
3. 保留角色 `.txt` 裡的 GIF 檔名與大小寫；不要任意改成 PNG 或重新命名。
4. GIF 請輸出為索引色圖。避免在邊緣留下半透明像素或抗鋸齒後混入粉紅色邊線。
5. 不要在純美術替換時修改動畫定義的 `Offset`、`BBox`、`attack`、`Delay`；這些定義控制站立位置、碰撞與攻擊判定。

## 交付前檢查

- 在圖片編輯器以吸管工具確認背景是 `#FF00FF`，而非近似色。
- 與原 GIF 比較畫布尺寸是否完全一致。
- 確認 GIF 可開啟、檔名與分鏡總表一致。
- 遊戲內測試 idle、walk、attack、jump、pain、fall；確認無洋紅方塊、角色沒有漂浮、攻擊判定未偏移。
